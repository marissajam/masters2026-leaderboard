# Masters Leaderboard – GitHub Pages package

## Publish to GitHub Pages
1. Create a new public GitHub repository.
2. Upload `index.html`, `.nojekyll`, and this `README.md` to the repo root.
3. In GitHub, go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** and **/(root)**, then save.
6. Wait a minute or two for the site to publish.

## Notes
- Auto-refresh is set to every 20 seconds.
- Data source order is Masters official first, PGA TOUR second, legacy backup last.
- The refresh badge shows the currently active source.
